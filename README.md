## 🔧 Como Executar o Projeto

1. Clone o repositório:
   ```sh
   git clone https://github.com/seu-usuario/cards.git
   ```
2. Acesse o diretório do projeto:
   ```sh
   cd cards
   ```
3. Instale as dependências:
   ```sh
   npm install
   ```
4. Execute o projeto:
   ```sh
   npm start
   ```
